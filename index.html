Circuitos lógicos

Compuertas AND, NOT, OR, NAND y XOR | Función: F = AB + C'




Esquema del circuito

Niveles del Circuito
Nivel 1

AND N1 = A · B

NOT N2 = ¬C

NAND Siempre produce 0 (D1=1, D2=1)

Nivel 2

OR N4 = N1 + N2

XOR N5 = NAND ⊕ D3 = 1

Nivel 3

AND F = N4 · N5 = N4

Nota: Como N5=1, la salida final es F = (A·B) + ¬C
F(A,B,C) = AB + C'

Tabla de Verdad

A B C N1=A·B N2=¬C N4=N1+N2 NAND D3 N5 F
Análisis de Resultados

Expresión en minterminos: F(A,B,C) = Σm(0,2,4,6,7)

F = A'B'C' + A'BC' + AB'C' + ABC' + ABC

Simplificación: F(A,B,C) = C' + AB

  • Si C=0: F=1 siempre
  • Si C=1: F=1 solo si A=1 y B=1
  • Activación en 5 de 8 casos (62.5%)

Aplicación: El circuito representa un sistema en base arduino donde C es un sensor PIR de seguridad, A y B son switches de autorización y F es la señal de activación (Para este caso un diodo Led).
